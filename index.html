Hosting an HTML Website on GitHub

This is how you can freely host your html website on the internet!

For Step by Step Instructions on How To Do This, Check Out These Videos:

First, you'll need to learn some basics about using GitHub. We'll start with this video - don't worry about what he says about Git, we're just going to focus on GitHub for now
Intro to GitHub (part 1)
Now, stop and make a GitHub account for your group (or if you want to be really professional, you can each make gitHub accounts, and then you can both collaborate on the same project

Next, this video will show you how to use GitHUb pages to host your html website for free
Intro to GitHub (part 8) - GitHub Pages
Once you've watched those 2 videos, you should be ready to made a repositiory for your website project, make a new .html file in GitHub and start writing html code right inside GitHub.
When you start working on CSS, you will make a .css file that you can reference in your html files
